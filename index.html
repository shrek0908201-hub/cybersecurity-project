Scope:

Phishing:
Fake emails, messages, or websites designed to trick users into giving passwords, bank details, or personal information. Always check the sender and never click suspicious links.
Malware:
Malicious software such as viruses, worms, trojans, and ransomware that can damage or steal data from your device. Install antivirus software and avoid downloading unknown files.
Firewall:
A security system that monitors and controls incoming and outgoing network traffic. It blocks unauthorized access while allowing safe communication.
Encryption:
A method of converting data into secret code so only authorized users can read it. Used in secure websites (HTTPS), banking apps, and messaging apps.
Two-Factor Authentication (2FA):
Adds an extra layer of security by requiring a second verification step like an OTP, fingerprint, or authentication app.
Social Engineering:
A technique where attackers manipulate people into revealing confidential information. Example: pretending to be a bank officer or tech support.
